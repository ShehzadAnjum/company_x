**Brand is LOCKED: "Veloce-AI"** (Italian *veloce* = fast). Supersedes the old "Aevum" placeholder.
Tagline **"Autonomous Intelligence. Always On."** **Logo (current, 2026-06-30): a blue/cyan
circuit-styled "Y/V" mark** (gradient chevron with PCB traces on the left stroke) + the VELOCE-AI
wordmark. **Canonical source: `resources/veloce_ai_final.png`** (full brand card) with extracted
transparent parts in **`resources/veloce_ai_web_assets_transparent_pngs/`** (`01_primary_logo`,
`04_app_icon`, `05_circular_mark`, 4 pillar icons, 6 swatches). *(This replaced both the chrome
infinity (∞) mark and Hamza's interim yellow "eloce" v0.2 wordmark — the whole site is now the
blue/violet spectrum, **no yellow**; see [[repo-and-access]] `hamza_yellow` tag for the prior cut.)*

**Official kit:** `resources/veloce_ai_brand_sheet/` — 28 labelled PNG parts (logo symbol, wordmark,
tagline, app-icon variants, the four "infinity concept" icons, colour swatches, type specs).
`resources/volace-ai.png` is the original flat sheet (note the typo'd filename). These are RGB on
dark; extract transparent assets by keying the near-black bg.

**Colours (current sheet):** cyan `#00F5FF` · blue `#4F8DFF` · violet `#7A5CFF` · purple `#B45CFF`
· near-black `#050816`. *(The earlier infinity-era palette was green `#00A693` / red `#ED1C24` / charcoal.)* **Type (per sheet):** primary **Sora / Orbitron / Exo 2**, secondary
**Raleway / Inter**. ⚠️ The live site does NOT yet use these fonts (it ships Plus Jakarta Sans +
Space Grotesk + Unigeo64 + JetBrains Mono) — aligning type to the sheet is an open follow-up.

**Live assets in `site/assets/` (2026-06-30 set, used across all pages):**
`veloce-logo-2026.png` (full lockup w/ tagline — hero masthead, footer, intro splash),
`veloce-word-2026.png` (lockup **cropped to drop the tagline** — nav wordmark on every page),
`veloce-icon-2026.png` (app-icon "Y" — nav compact symbol + final-card watermark), `veloce-mark-2026.png`
(circular mark), `favicon.png` (square app icon), and `ic-{speed,ai,agents,trust}.png` (the 4 brand-pillar
icons). The old `veloce-{symbol,lockup,logo,wordmark,tagline,yellow,v}.png` files are superseded/unused.
