**Brand is LOCKED: "Veloce-AI"** (Italian *veloce* = fast). Supersedes the old "Aevum" placeholder.
Tagline **"Autonomous Intelligence. Always On."** **Logo (current, 2026-06-28): a blue/cyan
circuit-styled "V" mark** (gradient chevron with PCB traces on the left stroke) + the VELOCE-AI
wordmark. Source: `resources/veloce-brand-img.png` and the cleaner pre-cropped set
`resources/veloce_ai_components/` (`01_main_logo_showcase.png` is the best lockup; `07_app_icon_ring_neon.png`
→ favicon). *(This replaced the earlier chrome infinity (∞) mark across the live site.)*

**Official kit:** `resources/veloce_ai_brand_sheet/` — 28 labelled PNG parts (logo symbol, wordmark,
tagline, app-icon variants, the four "infinity concept" icons, colour swatches, type specs).
`resources/volace-ai.png` is the original flat sheet (note the typo'd filename). These are RGB on
dark; extract transparent assets by keying the near-black bg.

**Colours (current sheet):** cyan `#00F5FF` · blue `#4F8DFF` · violet `#7A5CFF` · purple `#B45CFF`
· near-black `#050816`. *(The earlier infinity-era palette was green `#00A693` / red `#ED1C24` / charcoal.)* **Type (per sheet):** primary **Sora / Orbitron / Exo 2**, secondary
**Raleway / Inter**. ⚠️ The live site does NOT yet use these fonts (it ships Plus Jakarta Sans +
Space Grotesk + Unigeo64 + JetBrains Mono) — aligning type to the sheet is an open follow-up.

**Assets already extracted into `site/assets/`** (transparent, used on the live homepage):
`veloce-symbol.png` (nav small logo + final-card watermark), `veloce-lockup.png` (full symbol +
wordmark + tagline), `veloce-logo.png` (lockup **cropped to drop the tagline** — the hero masthead
logo), `veloce-wordmark.png`, `veloce-tagline.png` (the standalone tagline image below the hero logo),
`favicon.png` (circle badge), and `concept-{always,intelligent,autonomous,scalable}.png` (the
Infinity-Concept section icons). See [[showcase-site]] · [[identity-and-positioning]].
