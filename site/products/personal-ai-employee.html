spec What runs today vs. the roadmap

Running today

Verified in source — real sends, real OAuth.
  • NL → SQL → Excel report generator (genuine AI) — Gemini converts a plain-English email request into a report spec, queries the sales DB, and builds a multi-sheet, charted, totaled workbook.
  • Live integrations — Gmail send/receive (OAuth), WhatsApp via bridge, and Xero (P&L, invoices, contacts, create-invoice).
  • Kanban + human-in-the-loop approval — Inbox → Needs Action → Pending Approval → Done, with an audit log on every move.
  • CEO daily briefing & weekly audit — live Xero figures + vault stats composed into a styled email.
  • Cloud worker — drafts replies while you sleep; the local side approves and sends. Hardened multi-key LLM reliability layer underneath.

On the roadmap

Make the AI do the reasoning the role implies.
  • LLM-first triage — promote the existing LLM classifier over the rule engine so routing is judgment, not keywords.
  • RAG customer-service agent — answer Tier-1 inquiries from a per-tenant knowledge base, escalate the rest.
  • Multi-tenant + auth + billing — de-hardcode the single owner/brand; add metering and onboarding.
  • Social posting + bank feed — today scaffold-only / stubbed.

Honesty note: the live dashboard flow is ~70% deterministic rules, ~30% real LLM. The report generator is where the AI genuinely reasons — that's what we demo.

The role it replaces

The always-on back office.

Inbox triage, routine customer replies, and the analyst grind of pulling sales and finance reports — handled overnight, queued for one-tap approval. Works the full week, not forty hours.
